private function getDashboardStats(User $user)
{
    return [
        'total_books'    => Book::where('user_id', $user->id)->count(),
        'active_tasks'   => Task::where('user_id', $user->id)
                                ->where('status', 'pending')->count(),
        'active_habits'  => Habit::where('user_id', $user->id)
                                 ->where('active', true)->count(),
        'journal_entries'=> Journal::where('user_id', $user->id)->count(),
    ];
}

MODULE 02 — AI MENTOR

Talk to Your Books

Instead of highlighting a passage in Atomic Habits, ask James Clear directly — "How do I apply Chapter 1 to my schedule today?" The AI Mentor Chat turns reading into a conversation. 

// Streaming response with conversation history
public function chat(Request $request)
{
    try {
        return response()->stream(function() use ($request) {
            $response = $this->gemini->streamGenerateContent(
                $this->buildContextWithHistory($request->messages)
            );

            foreach ($response as $chunk) {
                echo "data: " . json_encode($chunk) . "\n\n";
                ob_flush();
                flush();
            }
        }, 200, ['Content-Type' => 'text/event-stream']);
    } catch (Exception $e) {
        // Fallback: graceful degradation
        return response()->json(['error' => 'AI temporarily unavailable'], 503);
    }
}
Engineering Challenge Solved

Maintaining authentic personas: Implemented strict system-level prompt constraints to prevent hallucination. When the AI breaks character or API fails, fallback mechanisms ensure graceful degradation — users never see broken functionality.
