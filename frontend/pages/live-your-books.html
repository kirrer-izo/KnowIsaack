Live Your Books

Live Your Books aims to bridge the gap between passive reading and the meaningful, real world application of knowledge. It leverages artificial intelligence to simulate mentorship from renowned thinkers and authors, guiding users towards continuous self-improvement.

The core modules are: Books, Task Generator, Habit Tracker, Journal, AI Mentor, and AI Features (It includes weekly growth summary, personalized advice, habit suggestions, and routine builder).

Project Purpose and Goal

My goal was to develop a self-improvement web application that helps users apply the lessons learned from books into their daily lives through interactive journaling and progress tracking.

I intended to bridge the gap between theory and practice, through modules like the AI Mentor that educated as tutors and renowned thinkers, so books become instrumental for personal growth.

There is also the AI Features module which enhances the user journey through personalized tasks tailored to the user, based on the books they read and the goals they accomplish.

ENGINEERING DEEP DIVE

Technical Challenges & Solutions

Overcoming engineering hurdles during development

Data Security & Privacy

Challenge: Protecting sensitive user data

Implemented field-level encryption for all journal entries. This ensures that user reflections remain private and secure at rest in the database.

AI Hallucination Management

Challenge: Ensuring AI reliability

Designed a transparent "Limitations & Risks" disclaimer model. This manages user expectations regarding the accuracy of AI-generated advice.

Prompt Engineering

Challenge: Inconsistent API responses

Refined system prompts to reduce strict Markdown formatting. Engineered a custom parser to render API outputs (lists, bolding) correctly in the UI.

Resource Optimization

Challenge: Server storage limits

Pivoted from full file uploads to a URI-based input system. This significantly reduced storage costs while maintaining sufficient context for the AI.
